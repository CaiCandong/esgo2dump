# esgo2dump
# dump elasticsearch with golang

---

- 当前仅支持 elasticsearch 7

---

### install

- with golang >= 1.18

  `go install github.com/loveuer/esgo2dump@latest`

- download pre-build release:

  [release](https://github.com/loveuer/esgo2dump/releases)

### usage

`esgo2dump -h`
